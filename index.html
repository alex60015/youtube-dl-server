youtube-dl

Enter a video url to download the video to the server. Url can be to YouTube or any other supported site. The server will automatically download the highest quality version available.

If you leave this input empty, the title of the video is taken instead.
The save directory is always where the administrator mounted the docker to. If you add a directory, it will be saved inside the mounted directory.

An error occured while downloading the file!
Please contact your administrator with the url to solve the problem.

The file will be downloaded in the background! You can now continue downloading more content.
